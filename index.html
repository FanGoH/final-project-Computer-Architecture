The CPU receives the following instructions 

  • ADD#: 0 0 0 0
                
  • SUB#: 0 0 0 1
                
  • AND#: 0 0 1 0
                
  • OR#:  0 0 1 1
                
  • LDA#: 0 1 0 0
                
  • MVA:  0 1 0 1
                
  • MVR:  0 1 1 0
                
  • ADD:  1 0 0 0
                
  • SUB:  1 0 0 1
                
  • AND:  1 0 1 0
                
  • OR:   1 0 1 1
                
  • NOP:  1 1 1 1
